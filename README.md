## 🧱 What I Believe In

```
SOLID principles aren't guidelines — they're the Jedi Code.
```

- 🏛️ **Clean Architecture** — Dependencies point inward. Always.
- 📖 **Clean Code** — If it needs a comment to explain, rename it.
- 🔧 **Design Patterns** — The right tool for the right job, never forced.
- 🧪 **Testing** — Untested code is legacy code.
- 🏕️ **Boy Scout Rule** — Leave the codebase cleaner than you found it.
